# Capsule-Network
3 parallel convolutional layered convolutional network, uses to classify that a CXR is a Covid-19 patient, Pneumonia and Normal.
In order to work the code we need:
keras=2.0.7
tensorflow-gpu=1.2
cuda=8.0
compiler gcc-5 and g++-5.
Link for dataset is :
